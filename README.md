# testProject

a [Sails](http://sailsjs.org) application

# Instructions

- Have Git, Node, Bower & Sails installed
- git clone https://github.com/dlvx/SailsTestProject.git
- cd SailsTestProject
- npm install
- cd assets
- bower install
- sails lift (In SailsTestProject, cd - || cd ..)
- go to http://localhost:1337/
